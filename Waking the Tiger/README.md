# Waking the Tiger - Trauma Healing Skills Gallery

A beautiful, meditative web app for exploring trauma healing skills and practices from Peter A. Levine's "Waking the Tiger: Healing Trauma."

## Features

- **24 Guided Skills**: Concepts, exercises, symptoms awareness, and principles
- **Audio Narrations**: Each card includes a guided voice script
- **Meditation Bell**: Calming bell sound for centering
- **Search & Filter**: Easily find specific skills and practices
- **Dark Theme**: Calming teal/sage color scheme
- **Responsive Design**: Works beautifully on all devices

## Technology

Built with vanilla HTML, CSS, and JavaScript - no frameworks needed.

## Local Development

Simply open `index.html` in your browser. No build process required.

## Deployment

This app is deployed on Vercel and ready to use immediately.

## Design Philosophy

Inspired by Muji minimalism and Dieter Rams principles - functional, calm, and beautiful.
